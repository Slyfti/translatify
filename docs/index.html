How it works

Three steps, and you're good to go!

  1. 01

    Install the extension from the Chrome Web Store or Firefox Add-ons.

  2. 02

    Open the popup and choose the language you want your lyrics translated into.

  3. 03

    Play any song with lyrics, and let Translatify translate them automatically as they scroll.

Simplicity first

No backend server,
no problems!

There is no backend server and no data collection. Translatify uses your own translation API key and stores everything locally: your language, your key, your cached translations.
